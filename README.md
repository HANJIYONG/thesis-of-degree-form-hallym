# thesis-of-degree-form-hallym

한림대 대학원 학위논문 latex 양식

```
└── thesis-of-degree-form-hallym
    ├── figure
    │   ├── sample.eps
    │   ├── sample.PNG
    │   └── sample.jpg
    ├── main.tex
    ├── hallym.sty
    └── reference.bib
```

### figure

jpg,png,eps 등 figure에 사용되는 파일들 저장 경로

### hallym.sty

main.tex에서 사용되는 각종 함수 및 style 선언

### reference.bib
